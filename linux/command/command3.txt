diff命令：
    功能：比较两个文件是否存在不同之处；
    使用：diff -param file1 file2
    参数：-B  不检查空白行；
         -c  显示全部内文，并标出不同之处；
         -H  比较大文件时可以加快比较速度
         -i  不检查大小写的不同
         -q  仅显示是否有差异，不显示具体差异信息
         -w  忽略全部的空格字符
         -y  以并列的方式显示异同之处
cmp命令：
    功能：用于比较两个文件是否一样
    使用：cmp -param file1 file2
    参数：-l 显示所有的不一样之处(显示十进制字符)
         -c 显示不同位置处的字符(显示十进制字符和char类型的字符)
         -s 不显示错误信息
         -v 显示版本信息
         -h 获取帮助信息
cksum命令：
    功能：对文件进行crc校验，用于校验两个文件是否一致；
    使用：cksum file1 file2
    参数：--help 获取帮助信息
         --version 获取版本信息

查看文件大小：
      ls命令 ：ls -l (文件大小以字节个数的形式显示)
      du命令 ：du -b file (以字节个数的形式显示)/du -h file(以人好识别的形式显示)
      stat命令 ：stat file (打印文件的详细信息，其中size项为文件大小)
      wc命令 ：wc -c file (参数-c表示统计文件中的字符个数)

查看目录大小：
      du命令 ：du -b dir(以字节个数的形式显示目录大小)
              du -h dir(以可读的形式显示大小)
              du -d dir(显示目录总计大小)
              du -a dir(显示目录占用的磁盘空间大小，还要显示其下目录和文件占用的磁盘空间大小)
              du -s dir(显示目录占用的磁盘空间大小，但是不显示其下子目录和文件占用的磁盘空间大小)
              du -l dir(统计目录占用磁盘空间大小时也会统计硬链接)
