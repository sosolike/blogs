file命令：
    功能：查看文件类型
    使用：file file_path;
    参数：
        -b : 忽略文件名
        -i : 打印文件mini类型
        -z : 解读压缩文件的内容
        -L : 直接显示符号链接所指向的文件的类型

find命令：
    功能：查找指定文件，任何参数之前的内容都会被当做查找的目录，如果不设置则默认在当前目录下查找文件；
    使用：find dir param
    参数：
        dir   指定需要查找的目录，如果不指明则认为在当前目录中查找；
        -name 使用文件名查找文件 
                find ./ -name a.txt
                find ./ -name a.\*
                find ./ -name \*.txt
        -size 使用文件大小来查找文件
                find -szie 10c  查找大小等于10个字节的文件或者目录
                find -szie -10c 查找大小小于10个字节的文件或者目录
                find -szie +10c 查找大小大于10个字节的文件或者目录
                find -szie -10k 查找大小小于10个kb的文件或者目录
                find -szie +10M 查找大小大于10个mb的文件或者目录
                find -szie +10G 查找大小大于10个gb的文件或者目录
                （c是字节(c小写)，k是KB(k小写)，M是MB(M大写)，G是GB(G大写)）
        -mtime 查找过去n天被修改过的文件
                find -mtime 2  查找文件更新日期距离现在1天以上，两天以内的文件
                find -mtime -2 查找文件更新日期距离现在2天以内的文件
                find -mtime +2 查找文件更新日期距离现在2天以上的文件
                find -mmin 2   查找文件更新日期距离现在1天以上，两天以内的文件
                find -mmin -2  查找文件更新日期距离现在2天以内的文件
                find -mmin +2  查找文件更新日期距离现在2天以上的文件
                (mtime指单位为天，mmin指单位为分钟)
        -empty 查找空文件或者空目录
        -perm  查找指定权限的文件或者目录
                find -perm 664 查找权限为664的文件或者目录；
                find -perm -u=664 查找文件或者目录所有者的权限为664的文件
                find -perm -g=664 查找文件或者目录用户组的权限为664的文件
                find -perm -o=664 查找文件或者目录其他用户的权限为664的文件
        -nouser 查找文件的用户id不存在的文件
        -nogroup 查找文件的组id不存在的文件
        -type 列出指定类型的文件
                find -type f 查找普通文件
                find -type d 查找目录
    
grep命令：查找文件内容
    grep常用格式 grep [参数] "模式" [文件]
    grep一共有三个:grep，egrep,fgrep
    参数：
        -i 忽略大小写
        -v 只打印没有匹配的，不打印匹配到的
        -n 显示行号
        -w 被匹配的文本只能是单词，不能是单词的一部分
        -c 显示一个由多少行匹配到了，不显示匹配到的内容
        -o 只显示被模式匹配到的字符串
        --color 将匹配到的内容以颜色高亮显示
        -A 显示匹配到的字符串所在的行以及其后N行
        -B 显示匹配到的字符串所在的行及其前N行
        -C 显示匹配到的字符串的前后n行；
        -r 在目录中查找
    模式：
        1：直接输入要查找的字符串，此时可以使用fgrep来代替grep提高查找速度；
        2：使用基本正则表达式
            .任意字符
            [abc] 表示匹配到abc其中一个字符
            [a-zA-Z] 表示匹配到一个a-zA-Z之间的字符
            [^123] 表示匹配一个除过123以外的字符
            \(sd\) 表示sd是一个整体
            | 表示或的关系
            ^表示锁定开头
            $表示锁定结尾
ln命令：为某一个文件在另外一个位置建立一个链接：
    软硬链接：
        软链接：
            1：软链接以路径的形式存在，类似于windows操作系统中的快捷方式
            2：软链接可以跨文件系统，硬链接不可以
            3：软链接可以对一个不存在的文件名进行连接
            4：软链接可以对目录链接
        硬链接：
            1：硬链接以文件副本的形式存在，但不占用实际空间；
            2：不允许给目录创建链接
            3：硬链接之呢在同一个文件系统中才能创建；
    使用：ln -param 源文件 创建的目标链接文件
    参数：-b 删除并覆盖以前建立的链接
         -d 允许超级用户制作目录的硬链接
         -f 强制执行
         -i 交互模式
         -n 把符号链接视为一般目录
         -s 软链接;
         -v 显示详细的处理过程
