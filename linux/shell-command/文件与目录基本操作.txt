目录相关操作：mkdir、rmdir、cd、pwd、ls/ll、cp、rm、mv

创建目录：mkdir [-m 权限值][-p] 待创建目录路径
    常用使用方式：
        创建目录：mkdir 待创建目录路径
            #默认方式创建目录，目录权限为系统默认权限，并且待创建目录的上级目录必须已经存在；

        递归创建目录：mkdir -p 待创建目录路径
            #可以使用递归的方式创建目录，目录的权限为系统默认权限，上级目录不存在的情况下会自动创建；
        
        创建指定权限的目录： mkdir -m 777 待创建目录路径
            #可以使用-m参数指定所创建目录的权限
            #如果-m和-p参数同时使用，并创建多级不存在的目录时，将会无视-m指定的权限，使用系统默认权限，如果两个参数都被使用，但是并未创建多级不存在目录时，被创建的目录权限则为-m指定的权限；

删除目录：rmdir [-p] 待删除路径(必须为空目录)
    常用使用方式：
        删除一个空目录:rmdir 待删除空目录路径

        递归删除空目录：rmdir -p 待删除空目录路径(多层目录都必须是空目录)

切换目录：cd 待切换路径
    常用使用方式：
        从根路径切换：cd /data/tmp
        从当前路径切换：cd ./data/tmp
        从上级路径切换：cd ../data/tmp
        切换至家目录：cd ~

查看当前路径：pwd [-P]
    常用使用方式：
        查看当前路径：pwd；
            #如果目录是一个其他目录的链接，该命令也只显示当前链接所处的路径；
            
        显示目录链接的真实路径：pwd -P；
            #如果目录是一个其他目录的链接，该命令可以显示链接的真实目录；
        
        示例：
            #在/tmp/lntest路径下执行以下命令:
                -- ln -s /tmp/origin
                -- cd ./origin
                -- pwd ：/tmp/lntest/origin  #显示链接的当前路径
                -- pwd -P ：/tmp/origin      #显示目录链接的真实路径

查看目录内容：ls [-a][-l][d]/ll
    常用使用方式：
        查看当前目录内容：ls

        查看当前目录内容，并且列出文件属性，权限等详细信息：ls -l

        查看当前目录内容，并且列出全部文件及目录(包括隐藏的文件以及目录)：ls -a或者ls -al

        查看当前目录内容，但是仅列出目录本身，而不是列出目录内的文件数据：ls -d

        ll查看当前目录内容，并列出详细信息；相当于ls -l


复制目录：cp [-asdilprsu] 源文件或者目录路径 目标文件或者目录路径
    参数意义：
        -p:连通文件的属性一起复制，而非使用默认属性
        -d:若源文件或者目录是一个链接，则复制链接而非文件或者目录本身
        -r:递归复制
        -a:等价于-pdr
        -f:强制复制
        -i:交互式复制，如果源文件或者目录已经存在，覆盖时会先询问动作的进行
        -s:复制为链接
        -u:如果目标文件比源文件旧，才会复制；
    
删除目录：rm [-ifr] 待删除文件或者目录路径
    参数意义：
        -i：互动模式，每删除一个文件都会发起询问
        -f:强制删除
        -r:递归删除，可以用来删除非空目录(危险操作)

移动目录：mv [-ifu] 源文件或者目录路径 目标文件或者目录路径
    参数意义：
        -i：互动模式，每移动一个文件都会发起询问
        -f:强制移动
        -u:如果目标文件存在，且比源文件旧才会移动；

文件相关操作：
创建新文件:vim 文件路径
    1:vim三种模式：
        命令模式：打开或者新建一个文件时，默认状态即为命令模式，所以输入内容均会当做命令，而非输入字符
        输入模式：编辑文档时的状态即为输入模式，所有输入内容会被当做文档的内容；
        底线命令模式：对文件本身进行管理操作；
    2：模式切换：
        命令模式：
            1：新建或者新打开一个文件时即为命令模式；
            2：在输入模式使用Esc按键即可进入命令模式；
            3：在底线命令模式使用ESC键即可进入命令模式；
        
        输入模式：
            1：在命令模式使用a/i/o即可进入输入模式；
            2：在底线命令模式，需要先进入命令模式，然后再进入输入模式；
        
        底线命令模式：
            1：在命令模式按 : 即可进入底线命令模式；
            2：在输入模式需要先退回命令模式，然后再进入底线命令模式；
    
    3：相关操作：
        命令模式下的相关操作：
            1：set number   :显示行号
            2：set nonumber :设置隐藏行号
            3：a            :进入输入模式,在光标之后输入内容
            4：i            :进入输入模式,在光标之前输入内容
            5：:            :进入底线命令模式
            移动类命令
            1：H            :光标移动到这个屏幕的最上方那一行的第一个字符
            2：M            :光标移动到这个屏幕的中间那一行的第一个字符
            3：L           :光标移动到这个屏幕的最下方那一行的第一个字符
            4：G           :移动到这个文件的最后一行
            5：nG          :移动到这个文件的第n行
            6：gg          :移动到这个文件的第一行，相当于1G
            7：n[enter]    :表示光标向下移动n行
            8：[end]       :移动到光标所在行的最后一个字符
            9：[Home]      :移动到光标所在行的第一个字符
            10：[ctrl+b]    :屏幕往后移动一页
            11：[ctrl+f]    :屏幕往前移动一页
            12：[n<space>]  :光标往右移动n个字符

            查找与替换：
            1：[/word]      :从光标处向下寻找名为word的字符串
            2：[?word]      :从光标处向上寻找名为word的字符串
            3：n            :代表重复前一个查找动作
            4：N            :与n相反，代表反向进行前一个动作
            5：[:n1,n2s/word1/word2/g]:n1和n2为数字，代表行号；该命名意为在n1和n2之间查找word1，并替换为word2;
            6：[:n1,$s/word1/word2/g]:n1为数字，代表行号；该命名意为在n1和文件末尾之间查找word1，并替换为word2;
            7：[:n1,$s/word1/word2/gc]:n1为数字，代表行号；该命名意为在n1和文件末尾之间查找word1，并替换为word2，并且在替换之前提示用户是否进行替换;

            删除：
            1：x            :删除当前光标后面的一个字符
            2：X            :删除当前光标前面的一个字符
            3：nx           :向后连续删除n个字符
            4：nX           :向前连续删除n个字符
            5：dd           :删除光标所在行
            6：ndd          :向下删除n行
            7：dnG          :删除光标所在行到第n行的所有数据
            8：dG           :删除光标所在行到最后一行的数据
            9：d$           :删除光标所在处到该行最后一个字符
            10：d0          :删除光标所在处到该行最开始的字符

            复制：
            1：yy           :复制光标所在行
            2：nyy          :复制光标所在行向下n行
            3：yG           :复制光标所在行到最后一行的所有内容
            4：ynG          :复制光标所在行到第n行的所有数据
            5：y0           :复制光标所在位置到该行的首字符
            6：y$           :复制光标所在位置到该行末尾的字符

            粘贴：
            1：p            :将已经复制的数据粘贴在下一行
            2：P            :将已经复制的数据粘贴在上一行
            3：u            :恢复前一个操作
            5：.            :重复上一个操作

        输入模式相关操作：
            1：a            :从当前光标位置的下一个字符插入
               A            :从当前光标所在行的最后一个位置插入
            
            2：i            :从目前光标所在处的前一个字符开始插入
               I            :从目前光标所在行的第一个非空字符开始插入

            3：o            :在目前光标所在行的下一行插入新的一行
               O            :在目前光标所在行的上一行插入新的一行
            
            4：r            :取代光标所在位置字符
               R            :一直取代光标所在位置字符，直到按下ESC为止；

            5：ESC           :退出输入模式，进入命令行模式
        
        底线命令模式：
            1：:w           :保存编辑的内容
            2：:w!          :强制写入该文件，和权限有关
            3：:q           :离开文件，并不保存修改
            4：:q!          :不想保存修改强制离开
            5：:wq          :保存后离开
            6：:x           :保存后离开
            7：:w filename  :将编辑的数据保存成另一个文件
            8：:n1,n2 w filename :将n1到n2的内容保存成filename这个文件
            9：set nu/number:显示行号
            10：set nonu/nonumber:隐藏行号

查看文件详情：
    1：ls -al 目标文件路径
    2：ll | 目标文件路径
    3：stat 目标文件路径 (此种方式可以查看文件的系统信息)
    4：file [-bizL] 目标文件路径
    5：du 文件路径、du -h 文件路径 (可以查看文件大小)
    6：wc -c 文件路径(可以查看文件中的字符个数)


比较文件内容：
    比较两个文件是否存在不同之处：diff [-cHiqwy] 文件1路径 文件2路径
        功能：比较两个文件是否存在‘不同之处’；如果没有不同之处将不会有任何输出；
        参数意义：
            -c 显示全部内容，并标出不同之处
            -H 比较大文件时可以加快比较速度
            -i 不检查大小写的不同
            -q 仅显示是否有差异，不显示差异具体信息
            -w 忽略全部的空格字符
            -y 以并行的方式显示异同之处
            -B 不检查空白之处

    比较两个文件是否一样：cmp [-lcsvh] 文件1路径 文件2路径
        功能：用于比较两个文件内容是否一样，如果一样不会有任何输出，如果不一样将会显示不一样的地方
        参数意义：
            -l 显示所有不同之处(显示十进制字符)
            -c 显示不同位置处的字符(显示十进制字符和char类型的字符)
            -s 不显示错误信息
            -v 显示版本信息
    
    获取两个文件的校验码是否一致(CRC校验)：cksum 文件1路径 文件2路径
        功能：查看两个文件的CRC校验码是否一致；

文件内容操作：
    1：查看文件内容:cat [-nbs] 文件路径
        功能：从第一行开始顺序打印文件内容

        参数意义：
            -n :由1开始对所有输出行编号
            -b :由1开始对所有输出行编号，空白行不编号
            -s :如果遇到两个及以上的空白行换成一行空白

        常用方式：
            查看单个文件全部内容：cat 文件路径

            查看单个文件全部内容，带上行号：cat -n 文件路径

            查看单个文件全部内容,带上行号，除过空白行：cat -nb 文件路径
            
            查看多个文件全部内容：cat 文件路径1 文件路径2

            将查看的内容输出到一个新的文件：cat 文件路径 目标文件路径
    
    2：查看文件内容:tac 文件路径 
        功能：和cat相反，从最后一行开始,反向打印文件内容，不再支持-nbs参数
        常用方式：
            反向查看文件内容：tac 文件路径
    
    3：查看文件内容，并显示行号：nl [-bnw] 文件
        功能：查看文件全部内容，并带上行号
        参数意义：
            -b a     :表示无论是否是空行，也同样列出行号
            -b t     :如果有空行，空的那一行不要列出行号(默认值)
            -n ln    :行号在屏幕的最左方显示
            -n rn    :行号在自己栏位的最右方显示，且不加 0 
            -n rz    :行号在自己栏位的最右方显示，且加 0 
        常用方式：
            

    4：查看文件前几行：head [-ncqv] 文件路径
        功能：查看文件开头的内容，默认是前10行；
        参数意义：
            -n m     :查看前m行
            -c m     :查看前m个字节
            -q       :隐藏文件名
            -v       :显示文件名

    5：翻屏查看文件内容:less 文件路径
        功能：如果文件内容过多一次显示出来不容一看，可以使用less每次显示一屏，然后翻屏查看；
        常用方式:
            less 文件路径
        操作方式：
            [Pageup]    向上翻页
            [Pagedown]  向下翻页
            空白键       向下翻页
            上键头       向上翻一行
            下箭头       向下翻一行
            /子串        向下搜索子串内容
            ?子串        向上搜索子串内容
            n           重复前一个搜索
            N           反向重复前一个搜索
            q           离开这个less程序
    
    6：翻屏查看文件内容:more 文件路径
        功能：如果文件内容过多一次显示出来不容一看，可以使用more每次显示一屏，然后翻屏查看；
        常用方式:
            more 文件路径
        操作方式：
            空白键       向下翻页
            b           向上翻页
            /子串        向下搜索子串内容
            ?子串        向上搜索子串内容
            n           重复前一个搜索
            N           反向重复前一个搜索
            q           离开这个less程序
            :f          立刻显示出档名以及目前显示的行数
    
    7：阻塞式查看文件末尾几行:tail [-n number] 文件路径
        功能：查看文件末尾几行内容，并阻塞式显示；如果文件内容有增加，同时也会打印添加之后的内容；
        常用方式：
            tail -n number 文件路径
    

    8：切割文件内容，并查看：cut [-dfbcn] 待切割文件路径
        功能：按行切割文件，并将切割完成的内容打印；
        参数意义：
            -d     使用定义的分割符来分割；
            -f     选取字段，和-d联合使用：使用方法有：
                    只取第N项：-f N
                    从第N项取值末尾：-f N-
                    取第N项至第M项：-f N-M
                    取前N项：-f -N

            -c     以字符为单位分割；字段选取方式和-f参数相同
            -b     以字节为单位进行分割；字段选取方式和-f参数相同
            -n     取消分割多字节，仅和-b联合使用；避免对多字节字符切割而导致出错；

    9：切割文件内容，并查看：split

    10：切割文件内容，并处理：awk
