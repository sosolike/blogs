1:C语言中将数字字符串转换为数字：
    double strtod(const char *nptr,char **endptr);
        功能：从左向右扫描字符串，获取字符串中的数字，并转换为浮点数；
        返回值：返回浮点数，若没有则返回NUll
        说明：第二个参数若endptr不是null,将会把最后一个数字字符的指针传回

    long int strtol(const char *nptr,char **endptr,int base);
        功能：将字符串转换为长整型，
        返回值：长整型值；
        参数：
            nptr:目标字符串，函数从左向右扫描nptr，遇上正负号或数数字开始转换，直到遇上非数字字符或者\0则结束转换；
            endptr:若nptr不为空，则将最后一个数字字符的字符指针返回；
            base:设置需要查找的数字的进制，默认当做10进制；

    unsigned long int strtoul(const char *nptr,char **endptr,int base);
        功能：将字符串转换为无符号长整型，
        返回值：长整型值；
        参数:
            nptr:目标字符串，函数从左向右扫描nptr，遇上正负号或数数字开始转换，直到遇上非数字字符或者\0则结束转换；
            endptr:若nptr不为空，则将最后一个数字字符的字符指针返回；
            base:设置需要查找的数字的进制，默认当做10进制；

    int atoi(const char *nstr) 
        功能：将数字字符串nstr转换为int整型；
        返回值：返回转换成功之后的整形值；
        参数：待转换的字符串；

    double atof(const char *nstr) 
        功能：将数字字符串nstr转换为double整型；nstr中可以包含正负号，小数点或者E(e)表示的指数部分；
        返回值：返回转换成功之后的整形值；
        参数：待转换的字符串；

    long atol(const char *nstr) 
        功能：将数字字符串nstr转换为long整型；
        返回值：返回转换成功之后的整形值；
        参数：待转换的字符串；
    备注：
        1:上述函数的转换规则均为：数从左向右扫描nptr，遇上正负号或者数数字开始转换，知道遇上非数字字符或者\0则结束转换;
        2:上述函数需要引入stdlib.h头文件；

2：将数字转换为字符串：
    将整数转换为字符串：
        1：与ANSI标准兼容的方法：
            sprintf(str,"%d",num):
                功能：sprintf功能在于可以将数字打印至字符串，包括整数和浮点数都可以；
                参数：num需要打印的数字，str需要存储转换后结果的字符数组；第二个参数是格式化字符；
                返回值：无
                备注：此时的str必须是以字符数组的方式声明，不能使用字符指针的方式声明；
        2：与ANSI标准不兼容的方法：(有可能提示函数undefined)
            void itoa (int n,char s[]);：
                功能：将整型转换为字符串；
                参数：n:代表需要转换的整数；s代表需要存储转换之后字符串结果的字符数组；
                返回值：无
            void ltoa(int n,char s[])：
                功能：将长整型转换为字符串；
                参数：n:代表需要转换的整数；s代表需要存储转换之后字符串结果的字符数组；
                返回值：无
            void ultoa(int n,char s[])：
                功能：将无符号，长整型转换为字符串；
                参数：n:代表需要转换的整数；s代表需要存储转换之后字符串结果的字符数组；
                返回值：无

            将浮点数转换为字符串：
                gcvt()：将浮点型数转换为字符串，并且四舍五入；
                ecvt()：将double类型的浮点数转换为字符串，转换结果不包含十进制小数点；
                fcvt()：指定位数为转换精度

3：转ASCII码：
    toascii(int c)
        功能：用来将整数转换为ascii码值；
        参数：需要被转换的整数；
        返回值：返回转换成功的ASCII码；
